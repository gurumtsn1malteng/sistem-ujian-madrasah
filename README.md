# sistem-ujian-madrasah
Sistem Ujian Online (Wrapper Google Form) dengan fitur deteksi pindah tab/minimize
# Panduan Sistem Ujian

Cara membuat link untuk siswa:
1. Ambil ID Google Form (kode acak di tengah link form).
2. Gabungkan dengan link website ini.

Format Link:
https://nama-situs-kamu.netlify.app/?id=[ID_GOOGLE_FORM]

Rumus Excel:
="https://nama-situs-kamu.netlify.app/?id=" & A1
